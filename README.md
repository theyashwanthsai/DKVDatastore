# KVDatastore
A simple implementation of a nosql "database" using go. Experimental, built for learning purposes

### Basic Idea:
```plaintext
-Database
    -User1
        -Cluster1
            -User1_Cluster1.json
        -Cluster2
    -User2
        -Cluster1
            -User2_Cluster1.json
```
